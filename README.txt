<h1>Quantitative Assessment of G7's Collaboration in Sustainable Development Goals</h1>

<h2>This repository contains the code for this project:</h2>

<ul>
  <li><strong>SDG_Project.Rproj</strong>: The R project file.</li>
  <li><strong>main_code_and_results</strong>:
    <ul>
      <li>Contains all figures for the manuscript and supplementary materials.</li>
      <li>Includes an Excel file named "SDG_Data" summarizing all results from this work.</li>
    </ul>
  </li>
  <li><strong>results_and_figures_used_in_manuscript</strong>:
    <ul>
      <li>Contains the R script and data used in this study.</li>
      <li><strong>indicators (raw data)</strong>: Raw data downloaded from the <a href="https://sdg-tracker.org" target="_blank">SDG Tracker</a> database.</li>
      <li><strong>sdg_data</strong>: Cleaned SDG indicators data from 1999-2020.</li>
      <li><strong>partial_true_direction_un</strong>: Data indicating the desirable directions used in the synergy analysis.</li>
      <li><strong>gross_synergy_markdown.Rmd</strong>: The R script for the analysis.</li>
      <li><strong>results</strong>: Outputs of the study derived from the R script (<code>gross_synergy_markdown.Rmd</code>).</li>
    </ul>
  </li>
</ul>
